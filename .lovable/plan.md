## الهدف
بناء **محرر ثيم** داخل لوحة الإدارة (`/admin/theme-editor`) شبيه بمحرر ثيمات Shopify: شريط جانبي للتحكم على اليمين + معاينة مباشرة للموقع في إطار (iframe) على اليسار، مع تبديل بين معاينة الجوال والديسكتوب، وزر "نشر" واحد في الأعلى.

## ما سيستطيع المدير فعله من المحرر

1. **اختيار الصفحة** (رئيسية / مفاضلة / برامج / كليات / من نحن / اتصل / فوتر / قائمة) من قائمة منسدلة بأعلى الشريط الجانبي.
2. **رؤية كل أقسام الصفحة** كقائمة في الشريط الجانبي (Hero / Intro / Announcements / Systems / Vision …).
3. **إظهار/إخفاء أي قسم** بزرّ عين 👁 بجانبه — يختفي فوراً من المعاينة ومن الموقع المنشور.
4. **توسعة أي قسم** لرؤية حقوله (عنوان، نص، صورة، رابط) وتعديلها مباشرة. الحفظ يُحدّث المعاينة فوراً.
5. **تبويب الألوان**: مُنتقي لون للـ Primary والـ Accent يُحدّث متغيرات CSS لكل الموقع.
6. **تبويب مكتبة الصور**: عرض كل الصور المرفوعة في bucket `site-images` كشبكة، انقر لنسخ الرابط أو حذف، رفع جديد.
7. **معاينة جوال/ديسكتوب**: زرّان أعلى الإطار يبدّلان عرض الـ iframe بين 390px و 1280px.
8. **نشر**: زر واحد "حفظ ونشر" بالأعلى — كل التعديلات تُحفظ دفعة واحدة وتظهر للزوار فوراً.

## التفاصيل التقنية

### قاعدة البيانات
- إضافة عمود `is_hidden boolean default false` إلى جدول `site_content`.
- مفتاح خاص `theme_colors` في `site_content` يحمل JSON في حقل `content` للألوان.

### الكود
- صفحة جديدة: `src/pages/admin/AdminThemeEditor.tsx` (مسار `/admin/theme-editor`).
- مكوّن جانبي: `SectionsPanel`, `ColorsPanel`, `MediaLibraryPanel`.
- خريطة ثابتة `PAGE_SECTIONS` تربط كل صفحة بقائمة `{key, label, fields[]}` — نفس مفاتيح `site_content` المستخدمة حالياً.
- كل مكوّن قسم في الموقع (HomeIntro, VisionSection, AnnouncementsBoard, SystemsCards, إلخ) يفحص `data[section_key]?.is_hidden` ويُرجع `null` إذا مخفي.
- المعاينة: `<iframe src="/?preview=1">`؛ المحرر يستمع لرسائل `postMessage` من الـ iframe، ويحقن CSS الألوان مباشرة عند التغيير عبر `iframe.contentWindow.postMessage`.
- بعد الحفظ، نستدعي `iframe.contentWindow.location.reload()` لإظهار التغييرات.
- مكتبة الصور: `supabase.storage.from('site-images').list()` + رفع/حذف.
- الألوان تُطبَّق globally بحقن `<style>` في `index.html` يقرأ من `theme_colors` عند الإقلاع.

### يبقى المحرر الحالي (EditableText/Image/Link داخل الموقع) كما هو
بحيث يبقى للمدير خياران: محرر سريع داخل الصفحة، أو هذا المحرر المركزي الكامل.

- إضافة/حذف بطاقات داخل الأقسام (تبقى عبر صفحات الإدارة الموجودة).
- محرر خطوط.
